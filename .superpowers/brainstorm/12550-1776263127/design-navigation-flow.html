Key Interaction Rules

  • All navigation is push-based — consistent with existing tools (Spiral Breathing, etc.)
  • Tab bar hidden on section detail editors, visible on main hub
  • Swipe-back gesture on main hub pops to Tools; disabled on section editors (use ✕ explicitly)
  • Save flow: ✓ tap → validate required fields → Firestore updateData → pop → refresh completion indicators
  • Cancel flow: ✕ tap → discard in-memory changes → pop (no confirmation dialog — data is short-form)
